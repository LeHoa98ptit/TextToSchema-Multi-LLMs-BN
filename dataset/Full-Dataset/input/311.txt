A company's workflow management system is as follows:
The system contains multiple organizations, and each organization can initiate multiple processes.
Process information includes Process ID, Parent Process ID (can be null), Organization ID, Process Type, Log Information, Resource ID, Status, Start Time, and End Time.
Each process may generate multiple events. Event information includes Event ID, Process ID, Event Type, Log Information, Status, and Timestamp.     
The system needs to record detailed information for each process and its related events to track the execution of workflows.
