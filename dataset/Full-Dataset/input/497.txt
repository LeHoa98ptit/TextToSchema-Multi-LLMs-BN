The database shall serve as a school sports event management system, managing information on various events, participants, results, and awards. It includes the following information:  

Records of participating students' basic information: Student ID, Name, Gender, Grade, Class, Contact Information  

Records of various school sports events: Event ID, Event Name, Event Type (such as running, long jump, throwing, etc.), Event Time  

Records of students registering for events: Student ID, Event ID, Registration Time  

Records of students' results in each event: Student ID, Event ID, Result, Event Date
