The system is a user-article sharing forum with the following features:
1.User Interaction Features:
Each user has a unique User ID, Username, Password, Registration Time, Last Login Time, and Role.
A user can participate in multiple activities, engage in video interviews, and comment or recommend books and articles.
2.Content Management:
The system manages several books and articles.
Book Information: Each book has a Book ID, Title, Author, and Description.
Article Information: Each article has an Article ID, Title, Content, Author, and Publish Time.
Activity Information: Each activity has an Activity ID, Activity Name, Activity Time, and Description.
Video Interview Information: Each video interview has a Video ID, Title, Video Link, and Publish Time.
3.Activity Management:
The system can organize source code reading activities, record activity names, times, and descriptions.
Users can participate in multiple activities.
The system also handles lottery activities, recording winner information and awarding gifts.
