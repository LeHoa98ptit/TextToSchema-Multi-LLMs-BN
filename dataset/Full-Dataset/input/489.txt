The database is a system for real-time communication or message push. The main functions include:

1. Application management: The system should manage different applications, each with a unique AppID and name.
2. Channel management: Manage channels in applications, support creation, closing, persistence, privatization, online status, and push notifications.
3. Client management: Manage client users of applications, each with a unique ID and user name.
4. Channel subscription: Manage channels subscribed by client users, ensuring that each client can only appear once in each channel.
5. Event logging: Record various events in the channel, including event type, timestamp, and payload content, and support quick query of events by channel ID and timestamp.
