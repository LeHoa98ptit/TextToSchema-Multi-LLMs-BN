Business Requirements Description for the Shared Bicycle Management System:
The shared bicycle company manages multiple users, each with a user ID, phone number, name, credit, and status.
The company manages multiple bicycles, each with a bicycle ID, location, status, and unlock code.
The company records multiple trips, each trip requiring the recording of user and bicycle usage information, including bicycle number, start time, end time, cost, and payment status.
The company manages multiple fault records, each fault involving the user number who encountered the fault, bicycle number, fault description, and reporting time.
The company also manages billing rules, each rule including the billing method, unit price, and minimum fee.
