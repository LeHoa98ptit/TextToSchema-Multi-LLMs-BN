A library management system needs to record the following information:
The library has many readers, and the reader information includes the reader number, name, email, phone number, registration date and the last information update time.
The library's collection includes the book number, title, author, ISBN number, publication date, current inventory, book classification and the latest data update time.
The borrowing process is: after registering, readers can query the book inventory through the system, record the borrowing date for each borrowing, update the return date when returning, and automatically calculate the overdue fine amount if the return is overdue.
The same reader can borrow different books multiple times, and each book is allowed to be borrowed by different readers in a cycle. Each borrowing is allowed to be renewed three times and the renewal number is recorded.
The system needs to update the book inventory status in real time. When a book is borrowed, the inventory is deducted, and the inventory is restored after the book is returned. The latest modification time of the corresponding data must be updated for each operation.
