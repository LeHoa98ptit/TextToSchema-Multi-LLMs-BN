This database is an internal employee management system for an enterprise

Main functions

1. Department management:
Add, delete, and modify department information
Query department list

2. Position management:
Add, delete, and modify position information, including position name and salary
Query position list, which can be filtered by department

3. Employee management:
Add, delete, and modify employee information, including name, position, and superior manager
Query employee list, which can be filtered by department, position, or superior manager
Generate employee hierarchy chart, showing each employee's direct superiors and subordinates
Calculate employee total salary, by department or position

4. Management relationship:
Manage the superior-subordinate relationship between employees
Query a certain employee's direct subordinates or direct superiors

5. Report generation:
Generate salary report for department and position
Generate detailed information report for employees
