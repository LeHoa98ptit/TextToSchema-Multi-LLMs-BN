The database is a data management system that is mainly used to record user information, data records and their operation history, and track user operations on data (such as creation, update, deletion, etc.). User storage system user information, including the user's basic identity and contact information. Store managed data records, including the record's name, content, creator, and the user information of the most recent operation. Record various operations performed by users on data records, including operation type, time, remarks, and operation content. Track and record each user's operation history on data records, including operation type, time, and operation content.
