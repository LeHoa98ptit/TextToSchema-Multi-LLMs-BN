The system shall record alumni personal information, including Alumni ID, Name, Employer, Job Title, Department, Class, Enrollment Year, and ID Number. The donation management module shall record detailed information for each donation, including Donation ID, Donating Alumni, Donation Time, Donation Amount, Donation Type (such as one-time donation or recurring donation), and Donation Usage (such as sponsoring individuals or funding projects). Additionally, the system shall record basic information for each donation project, including Project ID, Project Name, and Start and End Time. Each project involves multiple beneficiary units, recording the benefit percentage for each unit and the specific benefit amount obtained by that unit from the project. Through the management of this information, the system shall effectively track the allocation and usage of alumni donations.
