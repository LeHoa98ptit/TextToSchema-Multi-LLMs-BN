Business Requirements Description:
This is a task management system with the following main features:
1.Task Management:Create, assign, track, and complete tasks.Each task includes:Task ID, Task Name, Task Description, Task Deadline, and Task Progress.Task Type, Task Status, and Task Priority.Each task is assigned to a user.
2.User Management:Add, delete, and manage users.Each user has a unique User ID, Username, Password, and Email.Each user has a fixed role.
3.Role Management:Define and assign user roles.Each role includes:Role ID, Role Name, and Role Description.
4.Task Type Management:Define and manage different task types.Each task type includes:Task Type ID, Task Type Name, and Task Type Description.
5.Task Status Management:Track task statuses (e.g., To Do, In Progress, Completed), including the status name and description.
6.Task Priority Management:Set task priorities.Each priority includes:
Task Priority ID, Task Priority Name, and Task Priority Description.
