This is a Publishing House Book Management System. For book management, the system needs to manage multiple books, with each book containing information such as ISBN number, book title, publication date, and category. For author management, the system records multiple authors' information for each book, including author ID, name, email, and address. For editor management, the system maintains multiple editors' information for each book, including editor ID, name, email, and office location. For book relationship management, the system needs to record authors and editors for each book, including book ISBN number, author ID, and editor ID.
