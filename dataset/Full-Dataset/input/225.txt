Design and implement a database management system for an enterprise-level natural language processing (NLP) platform, which shall support:

Company management (store basic company information, authorization tokens and their hash values, record creation and modification timestamps),      

User management (store detailed user information, password salt values and their hash values, manage permissions and record creation and modification timestamps),

Request processing log recording (store request ID, user ID, status, timestamp, request data, compression results and probe information),

and User feedback management (store rating, comments and creation timestamp).
