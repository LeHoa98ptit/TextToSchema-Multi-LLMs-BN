The requirements analysis of a certain enterprise management system is as follows:
1. Employee management: including employee information, job assignment, performance appraisal, salary number, creation time, update time, etc. An employee can participate in multiple projects.
2. Supplier management: including supplier information, supplier relationship management, creation time, update time, etc. A supplier can supply multiple projects.
3. Project management: including project information, project progress management, creation time, update time, etc. A project contains multiple employees and can have multiple suppliers.
4. Attendance management: including employee number, employee clock-in, attendance record, creation time, update time, etc.
5. Salary management: including salary payment, salary calculation, creation time, update time, etc.
