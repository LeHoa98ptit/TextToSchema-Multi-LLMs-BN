Agricultural Ecosystem Monitoring System requires recording monitoring data for multiple ecosystems. Each ecosystem has a System ID, Name, Location, Type (such as "Forest", "Wetland", etc.), and Current Status (such as "Healthy", "Degraded", etc.). Each monitoring record includes System ID, Monitoring Date, Temperature, Humidity, Soil Moisture, Plant Growth, and Operator. Each operator has an Employee ID, Name, Gender, Contact Information, and Position.
