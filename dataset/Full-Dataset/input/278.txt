This is a Construction Company Safety Inspection Management System. For project basic information management, the system records information such as project ID, project name, project manager, construction start date, expected completion date, actual completion date, project budget, and project location. For safety inspection records, the system maintains information including inspection ID, project ID, inspection date, inspection content, inspection results, and inspector. For projects with unsatisfactory inspection results, the system conducts tracking management, recording information such as tracking record ID, project ID, tracking date, tracking content, and tracking results.
