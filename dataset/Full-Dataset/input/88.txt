The mail system manages multiple users, each of whom has a user ID, email account, name, password, and creation time. The system manages multiple emails, each of which has an email ID, sender ID, subject, body, status, sending time, and creation time. Each time an email is sent, the recipient information of the email needs to be recorded, including the recipient ID, whether it was successfully delivered, and whether it has been read. The system also supports the email attachment function, and each attachment has an attachment ID, attachment file name, attachment size, and attachment file storage path. Each email sent can be accompanied by multiple attachments, and whether the attachments were successfully delivered and whether they have been read is recorded. By recording information about users, emails, recipients, and attachments, the integrity and efficiency of the mail system are ensured.
