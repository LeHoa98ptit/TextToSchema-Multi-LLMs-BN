This is a Student Information Management System. The main functions of the system are as follows:
1.Student Information Management: Manages basic student information such as student ID, name, gender, school, department, major, and ID number.
2.Teacher Information Management: Manages teacher information, including teacher ID, name, gender, title, and contact number.
3.Course Information Management: Manages course details, including course name, hours, credits, etc.
4.Grade Management: Stores students’ performance data, including regular grades, experimental grades, mid-term grades, and final grades.
5.Permission Management: Manages student and administrator permissions, including user ID, username, password, and role.
Each teacher can teach multiple courses, and each course can be taught by multiple teachers.
