The e-commerce system business requirement description: A certain e-commerce system needs to manage users, products, shopping carts, and orders, covering the basic processes of registration, shopping, and order placement. The system requires storing user ID, username, password, and email information during user registration. The system provides multiple products and needs to record the product ID, name, price, inventory quantity, and a simple description for users to browse and select. The shopping cart records the correspondence between users and selected products, including the product quantity. After confirming the purchase, the user can submit an order. The order shall include the creation time, order status (such as pending payment, paid, etc.), and specific product details (product ID, quantity, unit price, etc.) for the system to perform subsequent processing.
