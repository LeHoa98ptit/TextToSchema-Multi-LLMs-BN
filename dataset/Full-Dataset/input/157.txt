System Requirements Function Description

Front-end Function
1: User Login: Freshmen log in through the WeChat applet. Each user needs to record the student number, student name, student gender, student class, student branch, and student dormitory.
2: View school information: Freshmen can view the school's introduction, facilities and other information. The school needs to record the school number, school name, school introduction, and school facilities.
3: View branch information: Freshmen can view the detailed information of their branch. The branch needs to record the branch number, branch name, and branch introduction.
4: View class information: Freshmen can view their own class information. The class needs to record the class number, class name, and class introduction.
5: View teacher information: Freshmen can view the detailed information of the class teacher. The teacher needs to record the teacher number, teacher name, teacher title, and teacher profile.
6: View course information: Freshmen can view the course information of their own class. The course needs to record the course number, course name, course description, and course credits.
7: View dormitory arrangement information: Freshmen can view their own dormitory allocation. Dormitory information includes dormitory number, dormitory building, dormitory room number, and number of dormitory beds.
8: View the course schedule: New students can view their own course schedule.

Background functions
1: Add school information: Administrators can add or modify the basic information of the school.
2: Add branch information: Administrators can add or modify branch information.
3: Add class information: Administrators can add or modify class information.
4: Add teacher information: Administrators can add or modify teacher information.
5: Add course information: Administrators can add or modify course information.
6: Add student information: Administrators can add or modify student information.
7: Add dormitory information: Administrators can add or modify dormitory information.
8: Add course schedule information: Administrators can add or modify course schedule information.
