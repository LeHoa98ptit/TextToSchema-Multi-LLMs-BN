An online learning platform's exam system is designed as follows:
The platform has multiple exams. Exam information includes Exam ID, Name, Time, etc.
Each exam contains multiple practice questions. Practice question information includes Question ID, Exam ID it belongs to, Question Content, Score, etc.
When a student takes an exam, the student submits answers. The submission record contains Submission ID, Exam ID, Student ID, Submission Status.    
A student has information such as Student ID, Name, etc.
Each submission has answers to multiple practice questions. The answer record contains Practice Question ID it belongs to, Submission ID it belongs to, Answer Content.
