System Requirements Function Description

1. Query Function
Site Query: Users can enter the site name, and the system will return all bus routes passing through the site and their related information.
Bus Route Query: Users can enter the name of the bus route, and the system will return all the stations and their related information along the route.
Station-to-Station Query: Users can enter the starting station and the terminal station, and the system will return all transfer routes from the starting station to the terminal station, with direct routes displayed first.

2. User Management Function
Registration: Users can register a new account and enter information including account number, password, name, phone number, etc.
Login: Users can log in to the system, divided into general users and administrator users.
Modify information: Registered users can modify their basic information.
Delete user: Administrator users can delete user information in the system.

3. Backstage Management Function
Route Management: Administrator users can add, modify, and delete bus route information.
Announcement Management: Administrator users can publish, modify, and delete bus announcement information.
Question Management: Administrator users can manage users' online questions, including answering questions and deleting questions.

4. Online question function
Post questions: Users can post questions or suggestions about bus routes.
Answer questions: Administrators or other users can answer posted questions.

5. Website announcement function
View announcements: Users can view bus news posted by administrators, including route changes, price adjustments, real-time traffic information, etc.
