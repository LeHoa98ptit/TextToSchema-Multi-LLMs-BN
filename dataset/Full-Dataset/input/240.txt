The database is a naval fleet management system. Its main functions are to manage information about ships, crew members, fleets, mission execution, and supplies. The system helps the navy efficiently manage the daily operations of the fleet, including ship management, crew task scheduling, fleet resource allocation, and ship supply status. The database contains the following functions:

Ship Management: records basic information about ships.
Fields: Ship ID (Primary key), Name, Type, Fleet ID, Construction Date, Service Status

Crew Member Management: records basic information about crew members.
Fields: Crew Member ID (Primary key), Name, Gender, Position, Fleet ID, Phone Number, Length of Service

Fleet Management: records basic information about fleets.
Fields: Fleet ID (Primary key), Name, Commander ID (Foreign key), Base

Fleet Mission Execution Management: records the status of missions executed by crew members and fleets.
Fields: Crew Member ID (Foreign key), Fleet ID (Foreign key), Execution Time, Remarks, Is Crew Member Primary Responsible

Ship Supply Management: records the supply status of ships.
Fields: Responsible Supply Crew Member ID (Foreign key), Ship ID (Foreign key), Supply Time, Supply Items, Supply Quantity
