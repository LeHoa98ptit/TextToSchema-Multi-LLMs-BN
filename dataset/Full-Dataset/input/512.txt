Business Requirements:
1.Customer: Each customer needs to record their basic information, including customer ID, name, ID number, and contact phone number.
2.Account: Each customer can have multiple accounts. Each account needs to record the account number, the customer ID associated with the account, and the account balance.
3.Transaction: Each account can perform multiple transactions. Each transaction record needs to contain a transaction ID, the associated account number, transaction type (such as deposit, withdrawal, etc.), transaction amount, transaction time, and transaction status. Each account can have multiple transactions, recording every transaction for each account.
