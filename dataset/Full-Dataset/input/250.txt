Business Requirement Description:
1.Frontend User Features:
Users can log in through the WeChat mini-program. Each user has a unique User ID, Username, Password, Phone Number, Registration Time, Last Login Time, and Role.
2.Product Management:
Administrators can add, edit, and delete product information. Each product includes Product ID, Product Name, Product Description, Price, Stock Quantity, Product Category, and Shelving Time.
3.Order Management:
Administrators can manage the order status, such as processing orders and shipping. Each order includes Order ID, User ID, Order Status, Total Amount, Order Time, Payment Time, and Delivery Address.
4.Purchase Management:
Each user can add multiple products to their shopping cart and place multiple orders, with each order containing multiple products.
