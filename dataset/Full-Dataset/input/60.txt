Research institutions conduct multiple scientific projects, each involving multiple researchers, with each researcher participating in only one project.
  Each project has a project ID, name, start date, end date, project leader, and budget. Researchers have an employee number, name, date of birth, gender, phone number, and position.
  Each project's research results must be recorded with a result ID, name, publication date, and researcher (1 person).
  The usage of project funds needs to be recorded with a fund ID, project ID, purpose, amount, and usage date.
