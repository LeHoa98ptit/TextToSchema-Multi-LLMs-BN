A machine learning model management and evaluation system is used to record and manage the operational status, performance metrics, hyperparameter settings, and dataset information of various machine learning models. The requirements analysis is as follows:
Model Run Records: Records the run ID, execution time, start time, end time, runtime, run path, maximum number of models, and model types for each model, implemented through the Leaderboard_Metadata table.
Model Performance Evaluation: Records the ID, name, performance metrics (such as RMSE, MSE, MAE, RMSLE), and whether the best family model of stacked ensemble exists for each model, implemented through the Leaderboard table.
Hyperparameter Management: Records the ID and name of hyperparameters, implemented through the Hyperparameter table, and records the hyperparameter values for each model, implemented through the Hyperparameter_Values table.
Model-Run Association: Records the association between each model and its run, implemented through the Model_Run table.
Dataset Management: Records the ID, name, owner, license, visibility, URL, creation date, last update date, data size, number of rows, and number of columns for each dataset, implemented through the Dataset_Metadata table.
Dataset-Run Association: Records the association between each dataset and its run, implemented through the Data_Map table.
Model Type Management: Records the ID and name of model types, implemented through the Model_Map table.
Hyperparameter Default Value Management: Records the default hyperparameter values for each model type, implemented through the Hyperparameter_Def_Values table.
Model Type-Model Association: Records the type of each model, implemented through the ID_Map table.
Dataset Variable Details: Records the ID, name, data type, number of unique values, and number of null values for each variable in the dataset, implemented through the Dataset_Variable_Details table, with each dataset's detailed information linked to a dataset.
Dataset Tag Management: Records the ID and name of dataset tags, implemented through the Tags table, and records the association between each dataset and its tags, implemented through the Tag_Map table.
