**Image Sending and Management System**

This system is designed to track the records of users sending images, manage basic information of the images, and provide logs and related links for sent images. Below is a detailed business description:

**Image Management:**

The system can manage the basic information of images, including Image ID, Image URL, image description, and compressed signature. Each image can be identified by a unique ID and can upload both its original and compressed version URLs along with description information.
This functionality enables the system to efficiently store and retrieve relevant data of images and supports image compression management to save storage space.

**User Management:**

The system can manage the basic information of users, including User ID, name, and phone number.
The user information helps to identify the sender of the images and is associated within the system to track the sending history of each image.

**Image Sending Records:**

The system can record every instance of image sending, including the sender, sending time, and corresponding Image ID. Each instance generates an independent sending link to track the specific sending situation of the image.
Through the sending records, the system can trace which user sent a particular image at a specific time and store the sending-related link.

**Sending Link Management:**

Each time an image is sent, the system generates a sending link and associates it with the sender, sending time, and image's ID.
This feature ensures that the sending of images can be tracked and managed, and provides a download link or access record for the image when needed.
