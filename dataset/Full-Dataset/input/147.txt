The key features of the system are as follows:
1.Class Management: Manage class information, including class name, department name, grade, etc.
2.Member Management: Manage the information of class members, including member ID, name, password, email, age, workplace, contact number, etc.
3.Announcement Management: Publish and manage class announcements, including announcement ID, announcement time, and announcement content.
4.Message Management: Manage class messages, allowing members to send messages. The system records sender ID, receiver ID, message content, send time, and read status.
5.Log Management: Record class logs, including log ID, log content, time, mood, etc. Each member can post their own logs.
6.Leave Management: Manage class messages, including message ID, message time, message content, and message count. Each member can leave messages.
7.Application Management: Handle applications submitted by class members, including application ID, application time, approval status, and application content.
