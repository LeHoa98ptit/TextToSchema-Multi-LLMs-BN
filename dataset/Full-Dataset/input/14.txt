  The Agricultural Irrigation Management System shall manage multiple irrigation systems, each capable of irrigating several plots. Each plot has a Plot ID, area, soil type, and current irrigation status (such as "Irrigated", "Not Irrigated"). Each irrigation system possesses a System ID, name, type (such as "Drip", "Spray"), installation date, and maintenance records. Each irrigation record includes System ID, Plot ID, irrigation date, irrigation duration, and irrigation amount.
