The database is an online forum or social media system where users can post and comment.

Main functions

1. User management
User registration: create a new user and record information such as username, email address, password, etc.
User login: verify username and password.
User information modification: allow users to modify personal information.
User activation status management: control whether users can use the system.

2. Post management
Post: users can post posts containing titles, descriptions, image paths, and thumbnail paths.
Post status management: control whether posts are visible.
Post browsing: users can view all visible posts.
Post search: search for posts by title or description.
Post deletion: users can delete their own posts, and administrators can delete any post.

3. Comment management
Post comments: users can comment on posts.
Comment browsing: users can view all comments on posts.
Comment deletion: users can delete their own comments, and administrators can delete any comment.

4. Data association
User-post association: record the author of each post.
User-comment association: record the author of each comment.
Association between posts and comments: Record all comments for each post.
